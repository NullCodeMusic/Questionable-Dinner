# Questionable Dinner
A free plugin for VCV Rack focusing on playful design.

### Fonts Used:

- **DSEG** by **Keshikan**

- **MuseoModerno** by **Omnibus-Type**

- **DIN 1451 Mittelschrift** by the **German Institute for Standardisation**, apparently

- **Terminal Grotesque** by **Raphaël Bastide**, with the contribution of **Jérémy Landes**. (velvetyne.fr)

- **Mess** by **Tezzo Suzuki** (velvetyne.fr)

- **Lithops** by **Anne-Dauphine Borione** (velvetyne.fr)

- **Departure Mono** by **Helena Zhang**

- **Voxel** by **Fine, Great. Studio**
